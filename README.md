# CRM System - LeadConnector Clone

A complete CRM system inspired by LeadConnector, built with Next.js, Supabase, and TailwindCSS.

## Features

- User authentication (Login/Signup)
- Dashboard with analytics
- Client management
- Lead tracking
- Messaging inbox with auto-labeling
- Calendar for appointments
- Accounts and payment tracking

## Technologies Used

- Next.js 14
- TypeScript
- TailwindCSS
- Supabase (Authentication & Database)

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Set up Supabase project and update environment variables
4. Run the development server: `npm run dev`

## Deployment

Deploy easily on Vercel with Supabase integration.
